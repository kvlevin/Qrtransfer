#### 工程简介

##### 本软件分 [pc端](./qrcode-generator-pyqt5-main) 和 [Android端](./zxing-android-embedded)

##### 软件目的是用于将电脑上文件通过二维码传输到手机端
#####     pc端使用pyqt5 图像框架
#####     Android 使用zxing二维码扫描库


#### 安装
##### Android端
    安装android-studio  
    导入项目

##### pc端
    安装 python
    新建 .venv 虚拟环境
    安装pyqt5
    安装qrcode
    执行 python main.py
